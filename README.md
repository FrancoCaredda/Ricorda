# Ricorda
Ricorda is a learning project focused on the object-oriented programming in C#.  
Ricorda is a console application that creates decks of terms in different languages. Each term is a pair of word and definition.

## How to build?
|Tools|Version|
|--|--|
|dotnet|10.0|
|Visual Studio|Code|

1. Clone the repository
2. Open the cloned folder in terminal
3. Run `dotnet build`
4. Run `dotnet run`
